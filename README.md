# QCustomPlot-PyQt5
Bindings for graphics lib QCustomPlot for PyQt5

## Requirements:
 - PyQt5 >= 5.0
 - sip
 - qmake (for Qt5)
 - libpythonx.x
 - make
 
## Build:
  python3 configure.py

## Install:
  Please libqcustomplot.so.1.0.0 to /usr/lib/pythonx/dist-packages/qcustomplot

## Check:
  in Python Interactive mode enter:
  import qcustomplot
  dir(qcustomplot)
